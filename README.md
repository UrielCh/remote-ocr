# remote-ocr

tesseract js via a nest-js + fastify server

## why

this project intend to debug tesseract.js random crash.

## Installation

```bash
pnpm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
